Instructions:
Turn: Left + Right
Accelerate: Up
Face to Decellerate: Down (turns ship to face away from movement)
Land: L (Click on planet to select, or "L" to select nearest planet)
Autopilot: A (turns towards current target)
Enter hyperspace: Z Target ship/planet/system: Click on it
Double-time: Slash key

debug: n to damage shields (energy is used to recharge shields, fire weapons, etc)

The "Load" button only does the very first part - loading a file from the system. It doesn't actually load anything from the file.
